# Next.js Integrations

This directory contains documentation and stubs for integrations available in the Next.js template.

## Available Integrations

### Better Auth + Google OAuth
- **Purpose**: Authentication with email/password and Google OAuth
- **Docs**: See `better-auth.md`
- **Environment Variables**: `BETTER_AUTH_SECRET`, `BETTER_AUTH_URL`, `GOOGLE_CLIENT_ID`, `GOOGLE_CLIENT_SECRET`

### Drizzle ORM + PostgreSQL
- **Purpose**: Database ORM with PostgreSQL
- **Docs**: See `drizzle-orm.md`
- **Environment Variables**: `POSTGRES_URL`

### OpenRouter / Vercel AI SDK
- **Purpose**: AI chat integration with OpenRouter
- **Docs**: See `openrouter.md`
- **Environment Variables**: `OPENROUTER_API_KEY`, `OPENROUTER_MODEL`

### Vercel Blob Storage
- **Purpose**: File storage abstraction
- **Docs**: See `vercel-blob.md`
- **Environment Variables**: `BLOB_READ_WRITE_TOKEN`
