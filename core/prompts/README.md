# Core Prompts

This directory contains shared, framework-agnostic prompt templates used by AI agents.

## Purpose

These prompts can be used across different frameworks (Next.js, Astro, etc.) for common tasks like:
- Feature implementation guidance
- Code review prompts
- Security analysis prompts
- Documentation generation

## Structure

Add prompt templates as markdown files with clear naming conventions:
- `feature-implementation.md`
- `code-review.md`
- `security-analysis.md`

## Usage

Reference these prompts in agent configurations or load them directly during agent execution.
