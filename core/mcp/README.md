# MCP Configuration

This directory contains the Model Context Protocol (MCP) server configuration used by AI agents.

## Files

- `mcp.json` - MCP servers configuration

## Usage

The MCP configuration defines what external services and tools the AI agent can access. Copy this file to `.mcp.json` in scaffolded projects.

## MCP Servers

Configure additional MCP servers in the `mcpServers` object. See the MCP documentation for available servers and their configuration options.
