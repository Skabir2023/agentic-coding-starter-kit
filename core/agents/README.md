# Core Agents

This directory contains reusable AI agent definitions that can be used across different frameworks (Next.js, Astro, etc.).

## Available Agents

- `coder.md` - General coding agent for implementation tasks
- `code-review.md` - Code review agent for PR feedback
- `security-scanner.md` - Security analysis agent
- `better-auth-expert.md` - Better Auth authentication expert
- `polar-payments-expert.md` - Polar payments integration expert
- `deep-dive.md` - Deep research agent for complex topics

## Usage

These agents can be loaded by Claude Code or other AI tools to provide specialized capabilities for specific tasks.

## Adding New Agents

Add new agent definitions as `.md` files with clear descriptions of the agent's capabilities, expertise, and usage patterns.
